== Description ==
This Plugin To Make Posts Filter With Ajax.
 Ajax Loading.
 Easy To Customize.
 Quick and easily access to information.


= Usage =

1. To get started, click on the menu y-data filter > Setting. 
2. Add 'div' to show results.
3. Add to widget.
4. Document included.

== Installation ==

1. Upload the 'ajax_data_filter' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Add the shortcode to your page.
